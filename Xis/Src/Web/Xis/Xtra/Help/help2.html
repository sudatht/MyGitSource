Postnummer

Bruk jokertegnet '%' dersom du ønsker å søke på f.eks. alle som er registrert med postnummer som begynner på '46' (Kristiandsand og omegn).
Skriv "46%".

Oppsigelsestid

Søker på alle som er registrert med ditt søke-valg eller kortere.
Eks
Velger du '2 måneder' får du frem alle som er registrert med 'ingen ', '14 dager ', '1 måned ' eller '2 måneder '.
